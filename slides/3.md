# Quick Setup

## Download and Install
1. Download this repository as ZIP or clone it
2. Create your slides in the `slides/` directory
3. Run `ruby build` to generate HTML files
4. Open `output/index.html` in your browser

## Deploy to GitHub Pages
1. Push your repository to GitHub
2. Go to repository Settings → Pages
3. Select "Deploy from a branch" → `main` → `/output`
4. Your presentation will be available online!
