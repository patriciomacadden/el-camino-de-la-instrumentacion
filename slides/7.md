# Adding Images and Media

## Step 1: Place Files
Put your images in the `slides/media/` directory:
```
slides/
  1.md
  2.md
  media/
    logo.png       ← Your image here
    diagram.jpg    ← Another image
```

## Step 2: Reference in Markdown
Use standard markdown syntax:
```markdown
![Description](/media/logo.png)
```

**⚠️ Always use `/media/filename` format**

![Demo Image](/media/image.png)
