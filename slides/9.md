# Custom HTML & Interactive Content

## HTML Elements
You can include custom HTML:

<div style="background: linear-gradient(45deg, #667eea 0%, #764ba2 100%); padding: 20px; border-radius: 10px; color: white; text-align: center;">
  <h3>Custom Styled Content</h3>
  <p>This is a custom HTML element with gradient background!</p>
</div>
